## Prompt

### Task
Implement a **static web application version** of the solo card game **Scoundrel** as a polished **portfolio project**. Deliver **fully working code**, separated into multiple files (**HTML, CSS, JavaScript,** and supporting modules if useful). The game should run **entirely in the browser** with **no external build step**.

### Context

#### Game Overview
Scoundrel is a solo **dungeon-crawl card game**. You manage: **health** (max 20), **weapons** (♦ 2–10), and **potions** (♥ 2–10). Your goal is to **survive against monsters (♣/♠)**. You **win** by clearing the dungeon deck; you **lose** if your health drops to 0. Your **score** is your remaining health if you win, or a **negative score** if you lose (sum of monster values left in the deck, subtracted from 0).
